# Gangsta Simulator Installer

This installer will set up the Gangsta Simulator game on your Windows computer.

## What's Included

- The main game executable
- All necessary dependencies
- SpeechCore library for screen reader support
- Empty Mods folder for custom mods
- Start menu shortcut

## System Requirements

- Windows 10 or later
- .NET 10.0 Runtime (installer will prompt if missing)

## Installation

1. Run the setup executable
2. Follow the installation wizard
3. Launch the game from Start Menu or Desktop shortcut

## Uninstallation

Use Windows "Add or Remove Programs" to uninstall the game completely.

## Mod Support

Place mod .dll files in the Mods folder to extend the game functionality.
